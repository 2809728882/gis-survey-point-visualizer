# Demo 演示流程

## 1. 打开系统

启动本地服务：

```powershell
python -m http.server 8000 -d frontend
```

浏览器访问：

```text
http://localhost:8000
```

打开后可以看到一个模拟施工场地，包括场地边界、测点、状态统计和测点清单。

## 2. 施工点位可视化

- 地图上默认显示 6 个模拟施工测点。
- 红色表示待放样，绿色表示已完成，黄色表示复核中，紫色表示风险点。
- 右下角清单同步展示编号、名称、状态和高程。

## 3. 坐标定位

- 点击地图点位或清单行。
- 左侧当前点位区域显示 WGS84 经纬度、局部施工坐标和高程。
- 地图弹窗显示状态、坐标和备注。

## 4. 导入外业数据

使用左侧“导入”按钮选择：

```text
frontend/data/sample_points.csv
```

导入后系统会重新生成地图点位、统计和清单。

## 5. 范围圈定

- 点击“圆形”后在地图上拖拽绘制施工范围圈。
- 点击“多边形”后依次点击范围边界，双击结束。
- 点击“范围”按钮清除已绘制范围。

## 6. 影像叠加

在“底图”下拉框中切换：

- 标准地图
- 卫星影像
- 地形参考

测点、边界和范围会叠加在底图之上。

## 7. 地名搜索与当前位置

- 在“定位”区域输入地名，点击“跳转”后地图会移动到搜索结果。
- 也可以输入 `经度,纬度`，例如 `121.23078,31.02545`，直接跳转到指定坐标。
- 点击“当前位置”后，浏览器会请求定位权限，并将地图移动到当前位置。
- 在高德底图下，搜索位置和当前位置显示会自动执行 WGS84 ⇄ GCJ-02 纠偏。

## 8. 进度监控

- 使用“状态”筛选待放样、复核中、已完成或风险点。
- 选中点位后可在“施工状态”下拉框中修改状态。
- 统计卡片和点位颜色会同步更新。

## 9. 数据导出

点击 JSON 或 CSV 按钮，可导出当前点位数据，用于后续归档、复核或二次处理。
