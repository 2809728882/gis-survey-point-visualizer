# 完整项目任务书

## 1. 项目名称

基于 GIS 的工程测绘点位可视化系统

## 2. 项目目标

这个项目主要是做一个施工测量点位管理 Demo，把外业测量数据从表格转换到地图上展示。目标不是替代专业测量软件，而是模拟现场常见的点位查看、坐标核查、范围圈定和进度状态管理流程，展示 GIS 前端和测绘数据处理的结合方式。

## 3. 应用场景

- 施工现场控制点、轴线点、放样点管理。
- GNSS/RTK、全站仪外业成果快速可视化。
- 施工区域范围圈定与进度状态核查。
- 无人机正射影像、卫星影像和地形底图叠加。
- 测量内业数据批量整理、转换和交付。

## 4. 功能模块

| 模块 | 任务描述 | 技术/工具 | 岗位价值 |
| --- | --- | --- | --- |
| 地图平台搭建 | 使用 OpenLayers 搭建二维工程地图，显示施工场地边界 | OpenLayers, HTML/CSS, JavaScript | 支持现场点位标注与施工可视化 |
| 测点标注与坐标定位 | 支持施工点位标注、坐标定位、范围圈定 | OpenLayers, GIS 坐标系统 | 提高现场放样精度，方便控制点布设 |
| 数据导入与格式转换 | 将 GNSS/RTK、全站仪数据导入平台并转换 | Python, CSV/Excel, GIS 坐标转换 | 适配施工内业数据，快速生成放样点 |
| 影像与地形整合 | 将无人机正射影像和地形数据叠加到平台 | OpenLayers, GIS 影像处理 | 辅助施工规划和地形研判 |
| 批量数据处理 | 编写脚本批量整理外业测绘数据 | Python, Pandas, NumPy | 提升整理效率，减少人工错误 |
| 线上查看与进度监控 | 支持浏览器查看测点和施工数据 | OpenLayers, Web 前端 | 现场快速核查点位，监控施工进度 |
| 轻量化部署 | 提供本地和服务器端部署方案 | Python, Web Server | 保障施工现场数据访问稳定 |

## 5. Demo 功能范围

当前 Demo 先做一条能跑通的核心链路，方便在本地浏览器演示：

- 打开浏览器后直接进入二维工程地图。
- 显示模拟施工场地边界。
- 显示内置施工测点。
- 支持 CSV/JSON 导入测点。
- 支持点击地图新增测点。
- 支持点击测点查看坐标和施工状态。
- 支持圆形和多边形范围圈定。
- 支持标准地图、卫星影像、地形底图切换。
- 支持按施工状态统计和筛选。
- 支持导出当前测点为 JSON 或 CSV。

## 6. 技术栈

- 前端：OpenLayers、HTML、CSS、JavaScript。
- 数据处理：Python、Pandas、NumPy。
- GIS：WGS84、Web Mercator、局部施工坐标近似转换。
- 可选扩展：QGIS、AutoCAD、无人机正射影像处理工具、PostGIS。
- 版本管理：Git + GitHub。

## 8. 实施计划

1. 需求分析：梳理施工测量与放样流程，确定点位字段、状态和范围圈定需求。
2. 地图平台搭建：使用 OpenLayers 创建二维地图并加载施工边界。
3. 点位可视化：实现测点加载、点击新增、状态样式和信息弹窗。
4. 坐标与范围：实现坐标展示、局部坐标转换、圆形和多边形圈定。
5. 数据导入：实现 CSV/JSON 前端导入和 Python CSV/Excel 转换。
6. 影像叠加：接入在线卫星影像、地形底图和后续无人机瓦片入口。
7. 进度监控：以点位颜色、统计卡片和状态筛选展示施工进度。
8. 轻量部署：提供本地、局域网和服务器部署说明。
9. 测试优化：验证导入、绘制、筛选、导出和移动端布局。

## 9. 验收指标

- 能在浏览器中显示施工地图、边界和测点。
- 能导入示例 CSV 并生成点位。
- 能点击点位查看编号、坐标、高程和状态。
- 能绘制施工范围并清除。
- 能按状态显示不同颜色并完成统计。
- 能通过 Python 脚本将 CSV/Excel 转为 JSON。
- 能在本地静态服务或直接打开 HTML 的方式演示。

## 10. 可扩展方向

- 后端服务：FastAPI + SQLite/PostgreSQL/PostGIS。
- 登录权限：项目管理员、测量员、施工员、监理查看权限。
- 数据版本：支持复测记录、误差对比和点位历史。
- 影像管理：支持无人机正射影像切片和 GeoTIFF 发布。
- 移动端：支持现场定位、离线缓存和扫码查点。
- 工程交付：自动生成放样单、成果表和点位报告。
